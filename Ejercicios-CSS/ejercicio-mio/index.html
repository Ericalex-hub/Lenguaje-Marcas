¿Para qué sirve esta página?

Esta página sirve para que la gente aprenda como comenzar en el mundo de la programación, a como se crean las páginas web y sobre todo a como poder páginas web de calidad.

Antes de empezar vamos a hablar sobre que es la programación, la programación es el proceso de diseñar, codificar, depurar y mantener un código de programación, ya sea HTML, CSS o JAVASCRIPT. Cada código de programación tiene su propio lenguaje para programar pero tamién cada una es diferente a los demas, pero la mayoría tiene una cosa en común y son las celdas.

Las celdas son pequeñas partes de la página que sirve para poner diferentes funciones, y cada celda que se habra se tiene que cerrar. Si no se cierran las celdas, dependeiendo del código que use puede o marcar error en toda la página o no mostrar nada.


¿Qué es HTML y para que sirve?

HTML es un código de programación que sirve para poder generar el aspecto de contenido de la página web, es decir, textos, imágenes, titulos, etc. Se podría crear una simple página web con un titulo y un texto o una página con encabezados, enlaces, videos, imagenes, etc. Todos los códigos de programación tiene su propio lenguaje de programación y HTML tambien tiene el suyo.

Por ejemplo, los más importante y básicos son los del titulo que se pone un h1 y que sirve para que el texto se vea más grande y en negrita, el h1 mientras el número de a lado aumenta, la letra se hara más pequeña, y luego para los parráfos se usa la p, que separa los textos en diferentes parráfos.

¿Qué es CSS y para que sirve?

Al igual que HTML este es otro tipo de lenguaje que va de la mano de HTML. CSS sirve para dar una mejor presentación a la página, es decir, que se puede poner distinto color a las letras, color de fondo, tamaños, etc. También otras cosas más complejas como bordes, sombras, animación, etc. Como CSS va de la mano de HTML, no se puede generar una página solo con CSS, es decir, que sino hay um código HTML y solo hay un CSS no se mostrará ninguna página, pero si solo hay HTML y no CSS si se mostrará la página pero sin ningún estilo, ya que HTML solo pone el contenido y CSS el estilo y color de la página.


Qué es JAVASCRIPT y para que sirve

Ahora que sabemos que HTML sirve para poner el contenido de la página y CSS se encarga del estilo y color de la página, nos falta darle la funcionalidad a la página, es decir, vamos a darle distintas funciones a la a distintas partes de la página.

JAVASCRIPT se encarga de todo eso, se encarga de darle función a un boton, un enlace, imagen, etc. Este tipo de código es muy diferente a los dos anteriores que hemos visto hasta ahora ya que JAVASCRIPT es un lenguaje de progrmación de alto nivel ya que este es muy distinto a los anteriores y para llegar a entenderlo nos haría falta un poco más de tiempo para que llegar a dominarlo por completo.
